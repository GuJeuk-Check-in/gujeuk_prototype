# Gujeuk 첫 방문 수기 등록부 프로토타입

태블릿 PC 화면에 종이 수기명부를 그대로 띄우고, 터치펜으로 칸 안에 직접 작성하는 정적 프로토타입입니다.

## 실행

별도 설치 없이 `index.html`을 브라우저로 열면 됩니다.

```bash
open index.html
```

로컬 서버로 보고 싶다면 아래처럼 실행할 수 있습니다.

```bash
python3 -m http.server 5173
```

그다음 `http://localhost:5173`으로 접속합니다.

## 프로토타입 범위

- 종이 명부처럼 날짜/방문시간/이름/생년월일/성별/전화번호/거주지/방문목적/동행인/개인정보 동의/비고 칸을 표시합니다.
- 날짜와 방문시간은 자동으로 채워집니다.
- 나머지 칸은 터치펜, 손가락, 마우스로 직접 필기합니다.
- 등록 완료 시 작성된 필기 이미지를 브라우저 `localStorage`에 저장합니다.

## 향후 서버 연동 방향

- 현재는 필기 원본 이미지를 저장합니다.
- 실제 연동 시에는 필기 이미지를 그대로 보관하거나 OCR로 기존 `POST /user/sign-up` 필드에 매핑할 수 있습니다.

실제 서버 연동 전 태블릿 수기 입력 흐름을 확인하기 위한 화면입니다.
