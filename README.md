# Gujeuk 첫 방문 수기 등록부 프로토타입

태블릿 PC 화면에 한 사람용 수기 등록지를 띄우고, 터치펜으로 큰 칸 안에 직접 작성하는 정적 프로토타입입니다.

## 실행

별도 설치 없이 `index.html`을 브라우저로 열면 됩니다.

```bash
open index.html
```

로컬 서버로 보고 싶다면 아래처럼 실행할 수 있습니다.

```bash
python3 -m http.server 5173
```

그다음 `http://localhost:5173`으로 접속합니다.

## 프로토타입 범위

- 한 사람만 작성할 수 있도록 방문 날짜/방문 시각/이름/나이/연락처/방문목적/개인정보 동의 여부/남자 동행인 수/여자 동행인 수 칸을 크게 표시합니다.
- 방문 날짜와 방문 시각은 자동으로 채워집니다.
- 이름, 나이, 연락처, 방문목적은 터치펜, 손가락, 마우스로 직접 필기합니다.
- 남자/여자 동행인 수는 `0`에서 시작하며 버튼을 누를 때마다 1씩 증가합니다.
- 등록 완료 시 작성된 필기 이미지를 브라우저 `localStorage`에 저장합니다.

## 향후 서버 연동 방향

- 현재는 필기 원본 이미지를 저장합니다.
- 실제 연동 시에는 필기 이미지를 그대로 보관하거나 OCR로 기존 `POST /user/sign-up` 필드에 매핑할 수 있습니다.

실제 서버 연동 전 태블릿 수기 입력 흐름을 확인하기 위한 화면입니다.
